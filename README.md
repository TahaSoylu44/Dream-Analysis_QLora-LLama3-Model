# Dream Catcher

*Ready to find out what your dreams are trying to tell you?*

This is my first hands-on fine-tuning project, built on top of **Llama 3 (8B)**. To train this model, I used **Unsloth** to accelerate GPU computations and fine-tuned it using the **QLoRA** technique. The model was trained on a custom synthetic dataset created in accordance with **Freudian** (psychoanalytic) and **Jungian** (analytical psychology) perspectives.

## Links
* **Model Weights (GGUF):** [Hugging Face - Dream Catcher Model](https://huggingface.co/MisterHolmes44/dream-catcher-llama3-GGUF)
* **Training Dataset:** [Hugging Face - Dream Catcher Dataset](https://huggingface.co/datasets/MisterHolmes44/dream-catcher-llama3-GGUF)

## Technical Details
* **Base Model:** Meta Llama 3
* **Fine-Tuning:** Unsloth & QLoRA
* **Prompt Format:** Alpaca Prompt Template
* **Format:** `.gguf` (Optimized for local CPU/GPU inference)
* **Backend:** FastAPI & Uvicorn

## Disclaimer
When you type in your dream, the model analyzes it and returns an interpretation. However, **this is not a real medical or psychological analysis**. The answers are generated by an AI and are not 100% scientifically accurate. This project is built purely for educational and experimental purposes.
